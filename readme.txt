== Description ==

This plugin will integrate Pelepay payment option into your wordpress blog / website easily and comfortably.
All you have to do after you installed the plugin is go to settings -> pelepay configuration and enter your pelepay email address.
This will create a price field on each post / page.

2 options to embed the payment buttons:

1. Once filled the price field, the payment button will appear on that page / post

2. Using shortcode to embed more than a single payment button
[pelepay price='xx' name='yyy']

Payments made to you by these buttons will appear in your pelepay account summary.

== Installation ==

1. Upload plugin folder 'pelepay' to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
